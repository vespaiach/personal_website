# Personal Blog

This is a personal blog built with Next.js and TypeScript.

## Getting Started

First, install the dependencies:

```sh
npm install
```

Then, run the development server:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Adding Blog Posts

To add a new blog post, create a new markdown file in the `docs` folder. The file should have the following format:

```markdown
---
title: "Your Post Title"
date: "2023-10-01"
---

Your post content goes here.
```
